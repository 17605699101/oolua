Detailed Description

Parameter traits for OOLUA::Proxy_class functions. These traits determine wether the parameters need to be passed back to Lua or not

Remarks
If parameter traits are not defined then it defaults to OOLUA::in_p. For function return values OOLUA::function_return is used (inside the generating function), this template is specialised for void types.
