OOLua is cross platfrom dependancy free library with uses C++03 to help with binding C++ code for use in Lua and also provides a simple to use typed interface for common operations in Lua.

This is not a fully original work, instead it builds on ideas from binding classes using Luna and Lua Technical Note 5. OOLua uses a domain specific language (DSL) set of macros which internally use templates and generate non intrusive proxies for types.

OOLua and the Lua stack

Contray to the name the Lua stack is not a stack yet OOLua treats it as one and provides two functions for normail interaction OOLUA::push2lua and OOLUA::pull2cpp

Lua compatibility

This version of the library is compatible with the following Lua implementations

  • Rio Lua 5.1 and 5.2 http://www.lua.org
  • LuaJIT 1.1.8 and 2.0 http://www.luajit.org/
